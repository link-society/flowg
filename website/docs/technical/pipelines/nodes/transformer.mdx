# Transformer

A transformer node references a [transformer](/docs/user/guides/transformers),
which will execute it on a log record and pass the result to the next nodes.

> **NB:** Transformer nodes references transformers by their unique name. If a
> transformer is updated, all transformer nodes referencing it will be
> invalidated.

## Example

import NodeTransformerFromJsonUrl from '@site/static/img/guides/pipelines/node-transformer-fromjson.png'

Let's assume we have a transformer named `from-json` with the following source
code:

