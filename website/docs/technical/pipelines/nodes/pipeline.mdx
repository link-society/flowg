# Pipeline

Pipeline nodes are used to pass log records through another pipeline, allowing
for modular and reusable pipeline designs.

> **NB:** Pipeline nodes references pipelines by their unique name. If a
> pipeline is updated, all pipeline nodes referencing it will be
> invalidated.
