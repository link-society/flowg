Deploying FlowG is easy:

Download and run the static binary:


tar xvf flowg-v0.46.0-linux-amd64.tar.gz
./bin/flowg-server

Or use a Docker container:


docker run \
  -p 5080:5080/tcp \
  -p 9113:9113/tcp \
  -p 5514:5514/udp \
  -v flowg-data:/data \
  linksociety/flowg:latest

An experimental Helm chart is available:


git clone https://github.com/link-society/flowg.git
cd flowg
helm install flowg ./k8s/charts/flowg -n flowg-system --create-namespace

It deploys and configure Fluentd to collect logs from all pods.
